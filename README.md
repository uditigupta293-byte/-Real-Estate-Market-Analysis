# -Real-Estate-Market-Analysis
📌 Project Overview

This project performs an Exploratory Data Analysis (EDA) on Gurugram's real estate market to uncover pricing trends, locality-wise insights, property type distributions, and factors influencing property prices.

The analysis focuses on understanding:

Most expensive properties and localities
Price distribution across property types
Ready-to-move vs under-construction pricing
Area vs price relationship
Price per square foot trends
Market patterns across different sectors and localities
📊 Dataset Features

The dataset contains information such as:

Price
Area
Rate per Square Foot
Property Type
Flat Type
Locality
Builder Name
Property Status
BHK Count
RERA Approval Status
Society Name

Identify the most expensive properties in Gurugram
Compare pricing across localities

📌 Key Insights
Sector 42 emerges as one of the most premium localities.
Independent houses and penthouses command the highest median prices.
Property prices generally increase with area.
Luxury properties create significant outliers in the dataset.
Price per square foot varies substantially across localities, indicating differences in demand and infrastructure.
